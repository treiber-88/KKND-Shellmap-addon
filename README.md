# KKND-Shellmap-addon

This is a enhanced shellmap for the original OpenKrush Kill Krush 'n' Destroy OpenRA mod, made by IceReaper, Dzieren, and others.

This shellmap was made by Dr. Mobius (thegrinningscrin) (Discord name)

All actors on the shellmap are original KKND game assets, which are owned by Ziggurat Interactive.

Special thanks to the OpenRA developers for making OpenRA free and open-source. I used the shellmap.lua from Red Alert 1 as the base for the Shellmap.lua for the enhanced shellmap for OpenKrush.
